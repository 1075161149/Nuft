# Nuft
Nuft is an C++17 implementation of the Raft protocol.

# Build
## Requirement
1. C++17 standard(e.g. g++ 7.2)
2. gRPC
3. [Nuke](https://github.com/CalvinNeo/Nuke), which is already packed in source
4. gtest

## Build tests
Build all tests by
```
make
```
Run all tests by
```
./test
```
